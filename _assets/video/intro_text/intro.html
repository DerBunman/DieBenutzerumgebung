Local installation process

This video demonstrates the installation process on a fresh Ubuntu 19.10.
Other Debian based distributions (eg Debian, Mint, ...) should be the same.

The installation performed in this video is the minimal configuration you can setup right now.
There are some more config options that are not in this video.
You can find these options when you read the playbook.yml and the inventory examples.
A detailed documentation will follow at some point.


Please note, that the install process is a bit slower than usual, because it runs in a VM and is being recorded at the same time.
Also my provider seems to have some issues, as there are some really slow downloads happening.
For example the Nerdfonts download came in with whooping 30KB/sec from github.

To spare you these tedious waiting times I have sped up some passages of the video.
These passages will have a visual indicator in the top right corner. (And also another fancy effect. So buckle your pants!)
